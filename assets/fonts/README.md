# Poppins Font

To use the Poppins font in this app, download the font files from Google Fonts:

https://fonts.google.com/specimen/Poppins

Download and place the following files in this directory:
- Poppins-Regular.ttf
- Poppins-SemiBold.ttf (weight 600)
- Poppins-Bold.ttf (weight 700)

Alternatively, you can remove the font declarations from pubspec.yaml and use the default system font.
